# C# Console Application - Basic Operations

## 📌 Description

This is a simple C# console application that demonstrates basic programming concepts such as output display, arithmetic operations, and user input handling.

---

## 🚀 Features

* Displays "Hello World" on the console
* Adds two predefined numbers (10 and 20)
* Takes input from the user
* Prints the entered input back to the console

---

## 🛠️ Technologies Used

* C#
* .NET Console Application

---

## 📂 Code Overview

The program performs the following steps:

1. Prints a welcome message ("Hello World")
2. Declares two integers and calculates their sum
3. Displays the sum
4. Accepts input from the user
5. Displays the entered input

---

## ▶️ How to Run

1. Open the project in Visual Studio
2. Build the project
3. Run using **Ctrl + F5**

---

## 💻 Sample Output

```
Hello World
Sum is: 30
Enter something: Hello
You entered: Hello
```

* Understanding basic C# syntax
* Working with variables and data types
* Using Console input/output
* Performing simple arithmetic operations
