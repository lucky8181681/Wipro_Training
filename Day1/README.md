# 📘 Day 1 - C# Basics

## 📌 Description

This folder contains basic C# console programs created as part of Day 1 practice.
The focus is on understanding fundamental concepts such as console input/output and simple operations.

---

## 🚀 Programs Included

### 🔹 1. User Input and Addition

* Prints "Hello World"
* Adds two numbers (10 and 20)
* Takes input from the user
* Displays the entered input

---

## 🛠️ Concepts Covered

* Console output (`Console.WriteLine`)
* User input (`Console.ReadLine`)
* Variables and data types
* Basic arithmetic operations

## 📂 Files

* `UserInputAndAddition.cs`
